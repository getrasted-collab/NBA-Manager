# NBA Manager UI Phase 4: Vertical Slice

The vertical slice connects three existing game experiences instead of creating mock screens: Dashboard, Player Profile, and Game Simulation/Result.

## Dashboard

- The franchise hero now includes the next opponent, venue, and date.
- The next-game action remains connected to the existing schedule route.
- Team leaders are interactive and open the selected player's profile.
- Existing franchise metrics, recent form, upcoming events, command center, inbox, and news remain intact.

## Player profile

- The existing player card is presented as a right-side management drawer.
- Morale, availability/injury, and fatigue are visible at a glance.
- Six core attributes include compact progress indicators.
- Existing season, career, contract, awards, injury, team, and draft data remain available.
- Adjust Role and Explore Trade connect to the existing Rotation and Trade routes.

## Simulation and result

- Existing Quick Sim and SimCast remain the simulation engines.
- SimCast receives a stronger broadcast scoreboard treatment.
- Completed matchup cards receive a final-result presentation using existing scores, quarter data, summaries, and factors.
- Existing player box-score rows continue to open the upgraded player profile.

No simulation rules, save format, route IDs, or statistical calculations were replaced.
